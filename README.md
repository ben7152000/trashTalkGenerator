# 幹話產生器 (Trash Talk Generator)

## 環境建置與需求 (Prerequisites)
1. Node.js : v14.15.4
2. Express v4.17.1
3. Nodemon 2.0.7
4. Eslint 7.17.0

## 安裝與執行步驟 (Installation and Execution)
1. 將專案複製到本機 (兩種方法)
> (1) 打開終端機輸入 
`git clone https://github.com/ben7152000/trashTalkGenerator.git`</br>
(2) 點選 download ZIP 下載

2. 進入專案資料夾安裝工具包
> 打開終端機輸入
`npm install`

3. 執行專案
> 打開終端機輸入 
`npm run dev`

4. 使用瀏覽器瀏覽</br>
>打開瀏覽器在網址列輸入 localhost:3000 即可瀏覽
